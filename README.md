# Dotfiles
This repo contains all of my personal configuration files.

## Prerequisites
- git
- nvim
- fzf
- ripgrep
- luarocks
- node
- npm
- zoxide
- rust
- kubectl
- kapp (gm only)

## Install
```bash
git clone https://github.com/ohmyzsh/ohmyzsh.git ~/.oh-my-zsh

chezmoi init lucas-it --apply
```
